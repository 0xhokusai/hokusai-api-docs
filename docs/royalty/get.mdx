### `GET /v1/nfts/{contractId}/{tokenId}/royalty`

Get royalty information of NFT.

### Request Parameter

|Parameter|Type|Example Value|Description|Required|
|--|:--:|:--:|--|--|
|`contractId`|string|`ADXS6zyen4NctAjH`|`contractId` issued by Hokusa API|Required|
|`tokenId`|string|`1`|`tokenId` issued by Hokusai API when minting NFT|Required|

### Response

|Key|Type|Example Value|Description|
|--|--|:--:|--|
|`receiver`|string|`0x3a93da588954acf4d0d8f1f1a4439fa79d84cf29`|Royalty recipient's Ethereum address|
|`percentage`|number|`10`|Royalty rate (0~100%)|

<TabItem value="js">

```js
const baseUrl = "https://api.hokusai.app"

const getRoyalty = async (baseUrl, apiKey, contractId, tokenId) => {
  const path = `/v1/nfts/${contractId}/${tokenId}/royalty`
  const url = new URL(baseUrl + path);

  const params = {
    key: apiKey
  };

  url.search = new URLSearchParams(params).toString();

  const res = await fetch(url);
  return res.json();
}
```

<TabItem value="py">

```py
import requests

base_url = "https://api.hokusai.app"

def get_royalty(base_url: str, api_key: str, contract_id: str, token_id: str):
  path = f"/v1/nfts/{contract_id}/{token_id}"

  params = {
    "key": api_key
  }

  res = requests.get(base_url + path, params=params)
  return res.json()
```
