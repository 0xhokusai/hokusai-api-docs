# ロイヤリティを設定する

### `POST /nfts/{contractId}/{tokenId}/royalty`

NFT のロイヤリティ情報を設定します。

### リクエストパラメータ
|パラメータ名|型|値|説明|
|--|--|--|--|
|`contractId`|string||使用しているコントラクトのEthereumアドレス|
|`tokenId`|string||ロイヤリティを設定したいNFTのEthereumアドレス|

### リクエストボディ
|キー|型|値|説明|
|--|--|--|--|
|`receiver`|string||ロイヤリティ受信者のEthereumアドレス|
|`percentage`|integer||ロイヤリティの割合（0~100%)|

### レスポンス
|レスポンス名|型|値|説明|
|--|--|:--:|--|
|`txHash`|string||トランザクションのハッシュ値|
