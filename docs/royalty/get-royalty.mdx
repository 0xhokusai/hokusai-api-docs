# ロイヤリティの情報を取得する

### `GET /nfts/{contractId}/{tokenId}/royalty`

NFT のロイヤリティ情報を取得します。

### リクエストパラメータ
|パラメータ名|型|値|説明|
|--|--|--|--|
|`contractId`|string||使用しているコントラクトのEthereumアドレス|
|`tokenId`|string||ロイヤリティを取得したいNFTのEthereumアドレス|

### レスポンス
|レスポンス名|型|値|説明|
|--|--|:--:|--|
|`receiver`|string||ロイヤリティ受信者のEthereumアドレス|
|`percentage`|integer||ロイヤリティの割合（0~100%)|
