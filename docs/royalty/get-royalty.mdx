# ロイヤリティの情報を取得する

### `GET /nfts/{contractId}/{tokenId}/royalty`

NFT のロイヤリティ情報を取得します。

### リクエストパラメータ
|キー|型|値の例|説明|
|--|:--:|:--:|--|
|`contractId`|string|`SJ6NeN6JsU1B0`|Hokusai API から発行された`contractId`|
|`tokenId`|string|`1`|Hokusai API により NFT を mint した際に発行された`tokenId`|

### レスポンス
|キー|型|値の例|説明|
|--|--|:--:|--|
|`receiver`|string|`0xdbc8cbd0a92ef715801d11d0100a1bffbed1e2d9`|ロイヤリティ受信者のEthereumアドレス|
|`percentage`|number|`10`|ロイヤリティの割合（0~100%)|
