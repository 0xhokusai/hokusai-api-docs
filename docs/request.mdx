Hokusai API requests follow the protocol below.
- All API requests should be sent over HTTPS.
- Parameters should be sent in the request body as a path, query, or JSON format.
- All API requests need a `key` parameter with the API key as the value in the query parameter. See [authentication](authentication) for more details.

## Request Body

Some endpoints requires passing parameters in the request body. The request body is in a JSON object format, and the name of the key is denoted by [the camel case](https://en.wikipedia.org/wiki/Camel_case).
An example of a request body is shown below.
```json title="/nfts/{contractId}/mint"
{
  "to": "0x640cd26d5bd53a2342bdcc8e9930282c542e0e98",
  "tokenUri": "https://example.com/1"
}
```
