# NFTの情報を取得する

### `GET /nfts/{contractId}/{tokenId}`

NFTの情報を取得します。

### リクエストパラメータ
|パラメータ名|型|値|説明|
|--|--|--|--|
|`contractId`|string||使用しているコントラクトのアドレス|
|`tokenId`|string||取得したいNFTのアドレス|

### レスポンス
|レスポンス名|型|値|説明|
|--|--|:--:|--|
|`id`|integer|||
|`tokenUri`|string||メタデータのURI|
