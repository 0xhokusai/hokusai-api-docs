# NFTの情報を取得する

`GET /nfts/{contractId}/{tokenId}`

NFTの情報を取得します。

### リクエストパラメータ
|パラメータ名|値|説明|
|--|--|--|
|`contractId`||使用しているコントラクトのアドレスです|
|`tokenId`||取得したいNFTのアドレスです|

### レスポンス
|レスポンスコード|説明|レスポンス|
|--|--|--|
|200|処理が正常に終了しました|`{"payee": "string",`|
|||`"commission": 100}`|
|400|誤ったパラメータが指定されました|Invalid input|
