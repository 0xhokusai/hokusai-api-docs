# NFTの情報を取得する

### `GET /nfts/{contractId}/{tokenId}`

NFTの情報を取得します。

### リクエストパラメータ
|キー|型|値の例|説明|
|--|:--:|:--:|--|
|`contractId`|string|`SJ6NeN6JsU1B0`|Hokusai API から発行された`contractId`|
|`tokenId`|string|`1`|Hokusai API により NFT を mint した際に発行された`tokenId`|

### レスポンス
|キー|型|値の例|説明|
|--|:--:|:--:|--|
|`id`|integer|`1`|発行した NFT に一意に紐づく`id`|
|`tokenUri`|string|`https://example.com/1`|メタデータのURI|
