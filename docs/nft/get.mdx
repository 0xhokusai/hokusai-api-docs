### `GET /v1/nfts/{contractId}/{tokenId}`

Get NFT Information

### Request Parameter

|Parameter|Type|Example Value|Description|Required|
|--|:--:|:--:|--|--|
|`contractId`|string|`ADXS6zyen4NctAjH`|`contractId` issued by Hokusai API|Required|
|`tokenId`|string|`1`|`tokenId` issued by Hokusai API when minting NFT|Required|

### Response

|Key|Type|Example Value|Description|
|--|:--:|:--:|--|
|`id`|integer|`1`|`id` uniquely associated with the issued NFT|
|`tokenUri`|string|`https://example.com/1`|URI of metadata|

<TabItem value="js">

```js
const baseUrl = "https://api.hokusai.app"

const getNft = async (baseUrl, apiKey, contractId, tokenId) => {
  const path = `/v1/nfts/${contractId}/${tokenId}`
  const url = new URL(baseUrl + path);

  const params = {
    key: apiKey
  };

  url.search = new URLSearchParams(params).toString();

  const res = await fetch(url);
  return res.json();
}
```

<TabItem value="py">

```py
import requests

base_url = "https://api.hokusai.app"

def get_nft(base_url: str, api_key: str, contract_id: str, token_id: str):
  path = f"/v1/nfts/{contract_id}/{token_id}"

  params = {
    "key": api_key
  }

  res = requests.get(base_url + path, params=params)
  return res.json()
```
