# NFTの情報を取得する

### `GET /v1/nfts/{contractId}/{tokenId}`

NFTの情報を取得します。

### リクエストパラメータ
|パラメータ|型|値の例|説明|Required|
|--|:--:|:--:|--|--|
|`contractId`|string|`ADXS6zyen4NctAjH`|Hokusai API から発行された`contractId`|Required|
|`tokenId`|string|`1`|Hokusai API により NFT を mint した際に発行された`tokenId`|Required|

### レスポンス
|キー|型|値の例|説明|
|--|:--:|:--:|--|
|`id`|integer|`1`|発行した NFT に一意に紐づく`id`|
|`tokenUri`|string|`https://example.com/1`|メタデータのURI|

<TabItem value="js">

```js
const baseUrl = "https://api.hokusai.app"

const getNft = async (baseUrl, apiKey, contractId, tokenId) => {
  const path = `/v1/nfts/${contractId}/${tokenId}`
  const url = new URL(baseUrl + path);

  const params = {
    key: apiKey
  };

  url.search = new URLSearchParams(params).toString();

  const res = await fetch(url);
  return res.json();
}
```

<TabItem value="py">

```py
import requests

base_url = "https://api.hokusai.app"

def get_nft(base_url: str, api_key: str, contract_id: str, token_id: str):
  path = f"/v1/nfts/{contract_id}/{token_id}"

  params = {
    "key": api_key
  }

  res = requests.get(base_url + path, params=params)
  return res.json()
```
