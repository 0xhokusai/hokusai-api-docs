# NFTを発行する

### POST
~~~
nfts/{contractId}/mint 
~~~
contractId Target contract ID  
tokenId TokenID of the NFT to get.

### Request body
~~~
{
  "tokenUri": "string"
}
~~~

### Responses
200
~~~
{
  "tokenUri": "string"
}
~~~
400  
Invalid input
