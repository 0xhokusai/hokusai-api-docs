# NFTを発行する

### `POST /nfts/{contractId}/mint`

NFT を [Mint](/glosarry#mint)します。

### リクエストパラメータ
|キー|型|値の例|説明|
|--|:--:|:--:|--|
|`contractId`|string|`SJ6NeN6JsU1B0`|Hokusai API から発行された`contractId`|

### リクエストボディ
|キー|型|値の例|説明|
|--|:--:|:--:|--|
|`to`|string|`0xdbc8cbd0a92ef715801d11d0100a1bffbed1e2d9`|NFTを発行するアドレス|
|`tokeuUri`|string|`https://example.com/1`|NFT metadata のURI|

### レスポンス
|キー|型|値の例|説明|
|--|:--:|:--:|--|
|`txHash`|string|`0x09e35eb7e41cd6d3759e0632d0f74f480b145173d611868371f0996c1a21d85f`|トランザクションのハッシュ値|
