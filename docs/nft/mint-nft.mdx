# NFTを発行する

### `POST /nfts/{contractId}/mint`

NFT を [Mint](/glosarry#mint)します。

### リクエストパラメータ
|パラメータ名|型|値|説明|
|--|--|--|--|
|`contractId`|string||使用しているコントラクトのアドレスで|

### リクエストボディ
|キー|型|値|説明|
|--|--|--|--|
|`to`|string||NFTを発行するアドレス|
|`tokeuUri`|string||NFT metadata のURI|

### レスポンス
|レスポンス名|型|値|説明|
|--|--|:--:|--|
|`txHash`|string||トランザクションのハッシュ値|
