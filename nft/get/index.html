NFTの情報を取得する

GET /v1/nfts/{contractId}/{tokenId}#

NFTの情報を取得します。

リクエストパラメータ#

パラメータ型値の例説明Required
contractIdstringADXS6zyen4NctAjHHokusai API から発行されたcontractIdRequired
tokenIdstring1Hokusai API により NFT を mint した際に発行されたtokenIdRequired

レスポンス#

キー型値の例説明
idinteger1発行した NFT に一意に紐づくid
tokenUristringhttps://example.com/1メタデータのURI

リクエスト例#

  • Curl
  • JavaScript
  • Python
curl "https://api.hokusai.app/v1/nfts/ADXS6zyen4NctAjH/1?key=fKe2rDFUZjSSdXfW"
const baseUrl = "https://api.hokusai.app"
const getNft = async (baseUrl, apiKey, contractId, tokenId) => {  const path = `/v1/nfts/${contractId}/${tokenId}`  const url = new URL(baseUrl + path);
  const params = {    key: apiKey  };
  url.search = new URLSearchParams(params).toString();
  const res = await fetch(url);  return res.json();}
import requests
base_url = "https://api.hokusai.app"
def get_nft(base_url: str, api_key: str, contract_id: str, token_id: str):  path = f"/v1/nfts/{contract_id}/{token_id}"
  params = {    "key": api_key  }
  res = requests.get(base_url + path, params=params)  return res.json()
