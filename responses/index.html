レスポンス

Hokusai API は HTTP ステータスコード、レスポンスボディによりリクエストの結果を返します。

レスポンスボディ#

レスポンスボディは、リクエスト結果の詳細を含むJSONオブジェクトです。 各種エンドポイントごとにレスポンスボディの内容は異なります。 以下にレスポンスボディの例を示します。

/nfts/{contractId}/mint からのレスポンスボディの例
{  "txHash": "string"}

HTTP ステータスコード#

Hokusai API では正常なリクエストに対するステータスコードとして、200を返します。 以下の表は、Hokusai API が返すステータスコードの一覧です。

HTTP ステータスコード説明
200リクエストが正常に処理されました
400必要なパラメータが無い、パラメータの値が間違っています
403誤ったエンドポイントにリクエストを送信しています
500内部でエラーが発生しています。誤ったtokenIdが送信されたり、認証がうまくできていない可能性があります
