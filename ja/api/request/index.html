リクエスト

Hokusai API のリクエストは以下のプロトコルに従います。

  • 全ての API リクエストは HTTPS で送信します。
  • パラメータは、パス、クエリ、または JSON 形式のリクエストボディで送信します。
  • 全ての API リクエストは、クエリパラメータに API Key を値として持つ key パラメータが必要となります。詳細は認証で確認してください。

リクエストボディ#

特定のエンドポイントは、リクエストボディでパラメータを渡すことができます。リクエストボディは、JSONオブジェクトの形で、キーの名前はcamel caseで表記します。 以下にリクエストボディの例を示します。

/nfts/{contractId}/mint へのリクエストボディの例
