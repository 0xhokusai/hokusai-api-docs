Hokusai API ドキュメントへようこそ！

Hokusai APIを利用すれば、あらゆる規模のNFTビジネスで、Hokusai API を使用して、NFTの発行、送信、二次流通手数料、独自コントラクトを利用し、暗号資産を必要とせずNFTビジネスを管理できます。 Hokusai では REST を利用した API の提供を行っています。このドキュメントでは、Hokusai API の仕様について説明します。
