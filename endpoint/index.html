エンドポイント

Hokusai API では以下のベース URL に対してリクエストを送信します。

https://api.hokusai.app/

また、Hokusai API では現在、以下のようなエンドポイントを提供しています。

エンドポイント説明
GET /v1/nfts/{contractId}/{tokenId}NFTの情報を取得する
POST /v1/nfts/{contractId}/mintNFTをMintする
GET /v1/nfts/{contractId}/{tokenId}/royaltyNFTのロイヤリティを取得する
POST /v1/nfts/{contractId}/{tokenId}/royaltyNFTのロイヤリティを設定する
