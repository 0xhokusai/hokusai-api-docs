ロイヤリティを設定する

POST /v1/nfts/{contractId}/{tokenId}/royalty#

NFT のロイヤリティ情報を設定します。

リクエストパラメータ#

パラメータ型値の例説明Required
contractIdstringADXS6zyen4NctAjHHokusai API から発行されたcontractIdRequired
tokenIdstring1Hokusai API により NFT を mint した際に発行されたtokenIdRequired

リクエストボディ#

キー型値の例説明
receiverstring0x3a93da588954acf4d0d8f1f1a4439fa79d84cf29ロイヤリティ受信者のEthereumアドレス
percentagenumber10ロイヤリティの割合（0~100%)。小数第二位以下は四捨五入されます。

レスポンス#

キー型値の例説明
txHashstring0xc968510337593d167efa433ab3726634cab8754d88f38e6e61e8ac7071601c42NFT を発行するトランザクションのハッシュ値

リクエスト例#

  • Curl
  • JavaScript
  • Python
curl -X POST -H "Content-Type: application/json" \ "https://api.hokusai.app/v1/nfts/ADXS6zyen4NctAjH/1/royalty?key=fKe2rDFUZjSSdXfW" \-d '{"receiver":"0x3a93da588954acf4d0d8f1f1a4439fa79d84cf29", "percentage": 10}'
const baseUrl = "https://api.hokusai.app"
const mintNft = async (baseUrl, apiKey, contractId, tokenId, receiver, percentage) => {  const path = `/v1/nfts/${contractId}/${tokenId}/royalty`  const url = new URL(baseUrl + path);
  const params = {    key: apiKey  };
  const requestBody = {    receiver,    percentage  };
  url.search = new URLSearchParams(params).toString();
  const res = await fetch(url, {    method: 'POST',    headers: {      'Content-Type': 'application/json'    },    body: JSON.stringify(requestBody)  });  return res.json();}
import requests
base_url = "https://api.hokusai.app"
def set_royalty(base_url: str, api_key: str, contract_id: str, token_id: str, receiver: str, percentage: float):  path = f"/v1/nfts/{contract_id}/{token_id}/royalty"
  params = {    "key": api_key  }
  request_body = {    "receiver": receiver,    "percentage": percentage,  }
  res = requests.post(base_url + path, params=params, json=request_body)  return res.json()
